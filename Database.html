Class: Database

new Database()

Implements the Singleton pattern to ensure a single database connection pool throughout the application. Uses environment variables for configuration: - PG_USER: PostgreSQL username - PG_HOST: Database host address - PG_DATABASE: Database name - PG_PASSWORD: Database password - PG_PORT: Database port number
